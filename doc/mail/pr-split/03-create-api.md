# PR 3 - 테스트용 메일 주소 생성 API

## 목표
- 직접 입력 또는 후보값 사용으로 테스트용 메일 주소를 생성하는 API를 제공한다.

## 포함 범위
- 생성 요청 DTO/검증 추가
- 생성 서비스 추가
- 생성 API 핸들러 추가
- 저장 직전 유니크 검증 추가
- OpenAPI 계약 추가

## 제외 범위
- 목록/상세 조회
- 수정/삭제
- 프론트엔드

## 문서 범위
- 생성 API 문서 추가

## 테스트 범위
- 직접 입력 생성 성공
- 중복 주소 생성 실패
- 잘못된 형식 실패
- 인증 사용자 소유 연결 검증

## 머지 조건
- 생성 API 하나만으로도 테스트용 메일 주소 자원을 안전하게 추가할 수 있어야 한다.
