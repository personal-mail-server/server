# PR 2 - 유니크 메일 후보 생성 API

## 목표
- 서버가 사용 가능한 테스트용 메일 주소 후보를 반환하는 API를 제공한다.

## 포함 범위
- 유니크 후보 생성 규칙 추가
- 후보 생성 서비스 추가
- 후보 반환 API 핸들러 추가
- OpenAPI 계약 추가

## 제외 범위
- 실제 테스트용 메일 주소 저장
- 목록/상세/수정/삭제 API
- 프론트엔드 UI

## 문서 범위
- 후보 생성 API 문서 추가
- 후보 생성 규칙 문서 초안 추가 가능

## 테스트 범위
- 후보 생성 성공
- 생성값 포맷 검증
- 충돌 회피 검증
- 인증 정책 검증

## 머지 조건
- 사용자가 생성 화면에서 후보값을 요청할 수 있는 안정적인 API 계약이 고정되어야 한다.
