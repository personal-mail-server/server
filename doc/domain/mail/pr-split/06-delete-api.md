# PR 6 - 테스트용 메일 주소 논리 삭제 API

## 목표
- 테스트용 메일 주소 삭제를 물리 삭제가 아닌 논리 삭제로 처리한다.

## 포함 범위
- 삭제 API 추가
- `deleted_at` 기록 처리
- 기본 목록 제외 규칙 연결
- OpenAPI 계약 추가

## 제외 범위
- 복구 기능
- 프론트엔드

## 문서 범위
- 삭제 API 문서 추가
- 논리 삭제 정책 문서 확정

## 테스트 범위
- 논리 삭제 성공
- 삭제 후 기본 목록 제외
- 타 사용자 삭제 차단
- 이미 삭제된 자원 재삭제 처리

## 머지 조건
- 삭제 정책이 물리 삭제가 아님을 코드와 문서 모두에서 명확히 보장해야 한다.
