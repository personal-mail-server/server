# PR 5 - 테스트용 메일 주소 수정 API

## 목표
- 기존 테스트용 메일 주소를 수정할 수 있는 API를 제공한다.

## 포함 범위
- 수정 요청 DTO/검증 추가
- 수정 서비스 추가
- 수정 API 핸들러 추가
- 저장 직전 유니크 재검증 추가
- OpenAPI 계약 추가

## 제외 범위
- 삭제
- 프론트엔드

## 문서 범위
- 수정 API 문서 추가

## 테스트 범위
- 수정 성공
- 자기 자신 값 유지 성공
- 타 자원과 중복 충돌 실패
- 삭제된 자원 수정 처리 정책 검증

## 머지 조건
- 생성과 별도로 수정 시 유니크 충돌 규칙이 명확히 검토되어야 한다.
