# PR 4 - 테스트용 메일 주소 목록/상세 조회 API

## 목표
- 사용자가 자신의 테스트용 메일 주소를 목록과 상세로 조회할 수 있게 한다.

## 포함 범위
- 목록 조회 API
- 상세 조회 API
- 소유권 검증
- 기본 목록에서 논리 삭제 자원 제외
- OpenAPI 계약 추가

## 제외 범위
- 생성/수정/삭제
- 프론트엔드

## 문서 범위
- 목록/상세 조회 API 문서 추가

## 테스트 범위
- 내 목록 조회 성공
- 내 상세 조회 성공
- 타 사용자 접근 차단
- 삭제 자원 기본 목록 제외 검증

## 머지 조건
- 읽기 흐름이 생성/수정/삭제와 독립적으로 검토 가능해야 한다.
