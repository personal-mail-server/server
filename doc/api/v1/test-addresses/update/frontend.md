# 테스트용 메일 주소 수정 프론트엔드 문서

## 문서 목적
이 문서는 `PUT /api/v1/test-addresses/{id}` 수정 API에 대응하는 프론트엔드 동작 기준을 정의한다.

---

## 기준 문서
- `doc/api/v1/test-addresses/update/backend.md`
- `doc/api/v1/test-addresses/frontend.md`

---

## 정리
수정 화면 동작의 상세 기준은 `doc/api/v1/test-addresses/frontend.md`를 따르며, 본 문서는 수정 API 엔드포인트 기준 폴더에서 프론트엔드 사양을 함께 찾을 수 있도록 유지한다.
