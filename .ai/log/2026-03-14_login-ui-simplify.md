## 변경 이유
- 기존 로그인 화면이 설명 영역과 장식 요소가 많아 일반적인 로그인 화면처럼 보이지 않았기 때문에 더 평범한 형태로 정리할 필요가 있었음

## 변경 대상
- `code/frontend/index.html`
- `code/frontend/styles.css`
- `.ai/log/2026-03-14_login-ui-simplify.md`

## 변경 내용
- 히어로 섹션과 과한 안내 블록을 제거하고 단일 로그인 카드 중심 레이아웃으로 변경
- 색상, 여백, 입력 필드, 버튼 스타일을 일반적인 로그인 화면에 가깝게 단순화
- 기존 로그인 동작과 검증 규칙은 유지

## 영향 범위
- 로그인 웹 화면
- 사용자 첫 진입 화면 인상
